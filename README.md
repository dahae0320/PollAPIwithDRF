# 설문조사 API
* [Building API with Django Rest Framework](https://books.agiliq.com/projects/django-api-polls-tutorial/en/latest/index.html)

### 실행 방법
1. git clone
2. venv 가상환경 설치 및 활성화
3. requirements.txt에 있는 패키지 설치
4. pollapi 폴더 진입, runserver


### 작업
* API 구현
* Test code
* CI

### Model
* User (사용자, django 기본)
* Poll (설문 조사)
* Choice (질문)
* Vote (투표)

### API 명세
* GET /users/
* POST /users/
